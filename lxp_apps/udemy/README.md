##How to generate Client Id and Client Secret for Udemy App Installation

1. Login to your udemy account and go to the url - https://www.udemy.com/user/edit-api-clients/
2. Click on "Request Affiliate API Client"
3. Provide name, description and website in the form (example for website - https://www.eightfold.ai)
4. Your request should be approved within 24 hours and after that your client credentials will be available on same page - https://www.udemy.com/user/edit-api-clients/
5. Use those credentials while installing the Udemy app on your account
